Prepare a healthier public software project.

OpenReady is a simple browser-based checklist for maintainers who want clear licensing, community docs, contributor guidance, and a project site that supports open collaboration.

Start checklist Read docs

What this project does

OpenReady helps project creators review the basic public materials that make an open-source software project easier to understand, reuse, and contribute to.

Project promise

This project does not sell hosting, support, consulting, or premium features. It exists as a free, open-source community tool.

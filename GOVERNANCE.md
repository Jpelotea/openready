# Governance

OpenReady is currently maintained by Joshua Carl Pelotea with support from community contributors.

## Decision-making

For now, the maintainer reviews issues and pull requests and makes final decisions based on:

- project purpose
- simplicity for beginners
- usefulness to open-source maintainers
- accessibility
- non-commercial alignment
- safety and maintainability

## Maintainer responsibilities

Maintainers should:

- keep the project non-commercial
- review issues and pull requests respectfully
- enforce the Code of Conduct
- keep documentation accurate
- protect the required Netlify attribution when hosted on Netlify's Open Source plan

## Future governance

If the project gains regular contributors, governance may be expanded to include additional maintainers and a clearer review process.

# Security Policy

## Supported versions

OpenReady is a small static browser app. The latest version in the main branch is the supported version.

## Reporting a vulnerability

If you find a security issue, please do not create a public issue with exploit details.

Instead, contact the maintainer privately:

```text
peloteajoshuacarl0@gmail.com
```

Please include:

- a short description of the issue
- steps to reproduce it
- potential impact
- suggested fix, if known

## Scope

OpenReady does not use a backend server, database, login system, or payment system. The main security concern is safe handling of browser-side input and exported checklist data.
